# DeFindex brand assets

Official logo SVGs from the DeFindex media kit
(https://docs.defindex.io/contact-us/media-kit). Use **exactly as provided** —
do not recolor, rotate, or otherwise modify them. Minimum size 40px.

Served from the site root, e.g. `/brand/logo-horizontal-light.svg`.

Only the light lockups are kept (the app sits on the light brand gradient).
Files are copied unmodified from the media kit and renamed to web-safe names
(the originals contain spaces and `#`, both unsafe in URLs):

| File | Source (media kit) | Colors | Use |
| ---- | ------------------ | ------ | --- |
| `logo-horizontal-light.svg` | Horizontal signature #01 | Lavender symbol + Dark Green wordmark, white bg | **Header — default** |
| `logo-vertical-light.svg` | Vertical signature #01 | Lavender symbol + Dark Green wordmark, white bg | Vertical lockup (optional) |
| `symbol.svg` | Symbol #03 | Dark Green symbol, transparent | Favicon / compact mark |

The horizontal/vertical lockups include a white background rectangle (part of
the official asset — left unmodified), so place them on a light/white or glass
surface where the white blends in. `symbol.svg` is transparent.

If a dark-background placement is ever needed, add a white-wordmark variant
from the media kit.
